# hello-world - Tracker

Created: 2026-02-04

## Directory Structure

- `TRACKER.md` - Main tracker file with all tasks and progress
- `plans/` - Detailed plan files for individual tasks
- `archive/` - Completed or cancelled tasks

## Usage

### View Progress
```
/review-tracker hello-world
```

### Add a Task
```
/add-task hello-world
```

### Update Task Status
```
/update-tracker hello-world --task=01 --status=Complete
```

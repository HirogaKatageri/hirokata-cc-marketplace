# Hello World Python

A simple, well-structured Hello World program in Python that demonstrates professional development practices.

## Description

This project implements a classic "Hello, World!" program in Python. While simple in functionality, it showcases proper project structure, version control, and documentation practices that scale to larger projects.

## Prerequisites

- Python 3.7 or higher

To check your Python version:
```bash
python --version
```

## Installation

1. Clone or download this project:
```bash
cd hello-world-python
```

2. (Optional) Create and activate a virtual environment:
```bash
# Create virtual environment
python -m venv venv

# Activate on macOS/Linux
source venv/bin/activate

# Activate on Windows
venv\Scripts\activate
```

3. Install dependencies (none required for this project):
```bash
pip install -r requirements.txt
```

## Usage

Run the program:
```bash
python main.py
```

Expected output:
```
Hello, World!
```
